# Compilation instructions
- To run this code, ensure you have Python and the psycopg2 library installed.
- To install requirements, run: pip install -r requirements.txt
# Usage
- Execute the script using Python: python a3q1.py
- Follow the on-screen menu to interact with the student database.
# Configuration
- Database connection parameters are read from a database.ini file.
- Ensure the database.ini file is correctly set up with your PostgreSQL credentials.
- The expected format of database.ini is as follows:
```
[postgresql]
host=localhost
database=A3q1
user=postgres
password=<<password>>
```
